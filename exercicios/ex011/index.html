Exercicio de links

Links são declarados no html com a tag <a src="link.com">aqui teriamos o link<a>


Links são declarados com o "a" de ancor. Possibilitando você anexar links internos e externos, links para downloads, em parte do seu texto, e até mesmo botões clicaveis.


Links externos

Links externos, são os links que levam para fora de seu site. É sempre importante usar dois parametroa para estes casos.

Parametros "taget= "_blank"" e "rel="external""


O parametro _blank faz com que seu site continue aberto para estes casos,emquanto abre uma nova guia para o site externo que esta clicando.

Ja o parametro external, ajuda com que sites de busca tenham informações de estar se tratando de um link externo para seu site

Links internos

Links internos dentro do proprio arquivo, são mais simples de tratar, você deve direcionar a pagina com um link basico que indica onde no sistema, esta o arquivo.

O parametro "rel" ainda é mantido, para indicar que se trata de uma pagina "next", enquanto os links que forem te direcionar de volta para sua pagina index, terão o rel "prev"

No follow

No follow é um parametro, muitas vezes você poder não vincular seu site a marcas ou sites externos para o dispositivo de busca.

Ele deve ser declarado dentro da tag "a" desta forma:rel="nofollow"

Isso protege que seu site seja vinculado muitas vezes a uma marca que você não sabe a vericidade, ou até mesmo a propagandas que pagaram por um patrocinio por exemplo.
