Listas ordenadas

Listas não ordenadas são declaradas com a tag <ol> dentro da tag de abertura, podemos colocar um parametro para formatar o mesmo. como por exemplo < ol type="a" start=4 >

Os parametros são diversos, e podemos mesclar outras listas dentro das mesmas listas e brincar. Misturando Ol e Ul juntas. Mas vamos deixar isso para depois.

Listas de definição

Nas listas de definição, usaremos como predefinição 3 tags novas:

<dt> para abrir a tag da lista de definição.

<dt> para abrir o termo da tag

<dd> para abrir a tag de descrição do termo acima

Linguagens que você precisa aprender para trabalhar com desenvolvimento de sites web

HTML
Linguagem de marcação de hypertexto. Usado para criação do "esqueleto" do conteudo da pagina web
CSS
Cascading Style Sheet, ou Folha de Estilo em Cascatas. Usado para criar a parte de estilos e desing do site.
JavaScript
Linguagem de programação para automatizar e da funcionalidade ao site. Como botões e configurações mais especificas de layout,e até mesmo de desing ao site.
