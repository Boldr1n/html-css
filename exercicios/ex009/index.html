Listas ordenadas

Listas não ordenadas são declaradas com a tag <ol> dentro da tag de abertura, podemos colocar um parametro para formatar o mesmo. como por exemplo < ol type="a" start=4 >

Os parametros são diversos, e podemos mesclar outras listas dentro das mesmas listas e brincar. Misturando Ol e Ul juntas. Mas vamos deixar isso para depois.
