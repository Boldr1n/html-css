Aplicação interna

A aplcação interna do código separado por uma tag de stype no heading, ajuda a organizar o código de forma que você possa alterar todas as tags de forma geral de uma unica maneira, veja o exemplo:


        <hegad>
            ...
            <style>
            h1{
                color: black; 
                background-color: cornflowerblue; 
                font-size: 2em;;
            }

            h2{
                background-color: cornflowerblue;
                 color: black; 
                 font-size: 1.5em;
            }
            
            p{
                color: darkblue; 
                font-size: 1.2em;
            }
            <tittle>

        </head>

Informações importantes

Quando aplicado a tag deste forma, você não deve declarar com as aspas duplas em seu código como no modelo anterior.
