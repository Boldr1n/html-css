Aplicação interna

Tags de CSS podem ser aplicadas de diversas maneiras, uma delas, e a forma menos eficiente, é a aplicação direta na tag. Veja o exemplo

Dentro da tag de HTML você deve adicionar o parametro "style" seguido das configurações de estilo para esta tag, como fontes, cores, tamanhos, alinhamentos, etc...

        <p style="color: darkred; font-size: 1.2em;>

        !</p>

Informações

Como ja dissemos, essa aplicação tag a tag pode ser problematica, apartir do momento em que se é necessario a aplicar mudanças e manutenções uma a uma nas tags. Chega a um ponto que se torna inviavel fazer este procedimento dessa forma em caso de sites maiores
